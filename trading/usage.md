# Trading Specialization Usage

## What This Is
This trading specialization is a structured advisor layer for Yue.
It helps with:
- market context
- narrative analysis
- opportunity screening
- setup analysis
- pre-trade checks
- post-trade review
- weekly review

It does **not** execute trades.

## Best Way To Use It
Use explicit mode-style requests in chat.

Examples:
- "Yue, use trading:narrative for crypto today"
- "Yue, use trading:screen for momentum setups"
- "Yue, use trading:pretrade on this chart"
- "Yue, use trading:analysis on BTC 4H"
- "Yue, use trading:review on this closed trade"
- "Yue, use trading:weekly for this week"

## Recommended Chat Setup
### Direct chat
Best for:
- serious trade planning
- post-trade review
- journaling
- market prep
- longer back-and-forth work

### Private Yue-only group with topics
Also good.
Use different topics for:
- market narrative
- opportunity screen
- live setup review
- post-trade review
- weekly review

That keeps discussions separated without clutter.

## Suggested Topic Structure
If you want clean organization, use topics like:
- Market Narrative
- Watchlist / Screens
- Live Setups
- Closed Trade Reviews
- Weekly Review
- Playbook Design

## Workflow Suggestion
### Before trading
1. ask for `trading:narrative`
2. ask for `trading:screen`
3. pick the best candidate
4. ask for `trading:pretrade`

### After trading
1. ask for `trading:review`
2. save lessons into journal / playbook later

### Weekend
1. ask for `trading:weekly`
2. update playbook if needed

## Important Rule
If you want structured output, explicitly say the mode.
That makes Yue lock into the right workflow faster.

## Shortcuts You Can Use
You do not need perfect syntax.
Any of these are fine:
- "use trading:narrative"
- "run trading:screen"
- "do a trading:pretrade"
- "review this with trading:review"
- "weekly review mode"

## If You Forget
Just ask naturally, for example:
- "Yue what’s the market narrative today?"
- "Yue screen crypto setups"
- "Yue check if this trade is valid"
- "Yue review this closed trade"

Yue should infer the mode from context.
