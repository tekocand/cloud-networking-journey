# Trading Research Routines

## Daily Routine
### 1. Market Context Pass
Do first:
- dominant narrative
- sentiment regime
- sector rotation
- event calendar risk

### 2. Opportunity Pass
Do second:
- narrative-aligned screen
- momentum scan
- breakout / pullback review

### 3. Setup Ranking Pass
Do third:
- compare top candidates
- reject weak names
- identify 1 to 3 real focus setups maximum

### 4. Pre-Trade Pass
Before entry:
- verify trigger
- verify invalidation
- verify target logic
- verify event timing
- verify risk size

## Weekly Routine
### 1. Market Review
- what themes dominated
- which sectors led / lagged
- what changed from prior week

### 2. Trading Review
- setups traded
- setups missed
- strongest idea type
- weakest idea type
- execution score

### 3. Playbook Update
- what setups still have edge
- what setups should be tightened
- where no-trade discipline failed

## Morning Brief Output
1. Market narrative
2. Sentiment regime
3. Rotation map
4. Top risk event today
5. Top 3 watchlist candidates
6. What to avoid

## Weekend Review Output
1. Weekly narrative summary
2. Sector/theme winner and loser
3. Best setup class
4. Worst setup class
5. Smart money alignment summary
6. Trading lesson for next week

## Rule
Routine outputs must be concise, structured, and decision-useful.
No motivational fluff.
