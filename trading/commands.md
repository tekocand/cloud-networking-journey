# Suggested Trading Commands

These are not shell commands.
They are recommended chat phrases for calling the trading system cleanly.

## Core Calls
- `trading:narrative`
- `trading:screen`
- `trading:analysis`
- `trading:pretrade`
- `trading:review`
- `trading:weekly`

## Examples
- `Yue, trading:narrative for crypto today`
- `Yue, trading:screen for breakout setups`
- `Yue, trading:analysis on BTC 4H`
- `Yue, trading:pretrade on this setup`
- `Yue, trading:review on this closed trade`
- `Yue, trading:weekly for this week`

## Group Topic Use
Suggested one-topic-per-mode pattern:
- narrative topic -> `trading:narrative`
- screen topic -> `trading:screen`
- live setup topic -> `trading:pretrade` or `trading:analysis`
- review topic -> `trading:review`
- weekly topic -> `trading:weekly`

## Fallback Rule
Natural language is still okay.
These command-like phrases just make activation cleaner and more repeatable.
