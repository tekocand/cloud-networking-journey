# Trading System Integration Plan

## Current State
The trading system now supports:
- market narrative
- screening
- setup review
- post-trade review
- weekly review

## New Source Strategy
### Primary Source
- X / Twitter for narrative discovery

### Secondary Sources
- news / catalyst pages
- charts / TradingView
- exchange or market dashboards
- macro / earnings calendars

### Internal Source
- trade journal
- playbook
- weekly review notes

## Recommended Usage Sequence
1. `trading:narrative`
   - start with X and other fast sources
2. `trading:screen`
   - narrow candidates
3. `trading:analysis`
   - deep dive best candidate
4. `trading:pretrade`
   - final check before entry
5. `trading:review`
   - after close
6. `trading:weekly`
   - weekend learning loop
