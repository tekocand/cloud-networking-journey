# Screening Framework

## Purpose
This framework turns broad market intelligence into specific opportunity selection.

The advisor should screen for opportunities systematically instead of jumping straight to random symbols.

## Screening Types
### 1. Narrative Screen
Find assets aligned with a strong current theme.
Examples:
- AI tokens
- Layer 2s
- energy stocks
- defensives
- rate-sensitive growth

Filter by:
- theme relevance
- liquidity
- trend quality
- catalyst clarity

### 2. Momentum Screen
Find assets already showing strength.
Check:
- trend direction
- relative strength
- volume expansion
- follow-through quality
- whether move is extended or still actionable

### 3. Breakout Screen
Find assets in clean compression / consolidation.
Check:
- base quality
- resistance level
- breakout trigger
- volume confirmation
- failure level

### 4. Pullback Screen
Find assets in strong trend but retracing into a valid area.
Check:
- trend quality
- pullback depth
- support quality
- reclaim trigger
- invalidation

### 5. Value / Mispricing Screen
Use carefully.
Applicable mostly for longer timeframe equities or structural trades.
Check:
- why mispricing exists
- catalyst needed to unlock value
- why market may still be right

### 6. Relative Opportunity Screen
Compare multiple candidates and rank them.
Rank by:
- context alignment
- setup clarity
- invalidation quality
- R profile
- event risk
- liquidity / execution quality

## Scoring Model
Use a simple score out of 10 based on:
- narrative alignment
- technical quality
- risk-reward quality
- catalyst clarity
- event risk penalty
- liquidity quality

## Output Structure For Screen Results
For each candidate:
1. Symbol / market
2. Theme / narrative fit
3. Why it qualifies
4. Setup type: breakout / pullback / momentum / mean reversion / value
5. Trigger
6. Invalidation
7. Target logic
8. Estimated R profile
9. Main risk
10. Match score / 10
11. Verdict: actionable / watchlist / reject

## Ranking Rule
The advisor should rank candidates in order and explain why #1 beats #2.

## Rejection Rule
If no candidate is good enough, say no valid screen result.
That is better than forcing a list.
