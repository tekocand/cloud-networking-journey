# Multi-Source Narrative Prompt

Use this when combining X with other sources.

## Prompt
Act as a disciplined market-intelligence analyst.
Use X as the primary source for narrative discovery, then use secondary sources for validation.
Do not confuse online attention with confirmed opportunity.

Inputs:
- X narrative notes:
- Market/news notes:
- Calendar/event notes:
- Chart confirmation notes:
- Flow / smart money notes:

Output exactly in this structure:
1. Dominant narrative
2. Secondary narrative
3. What X is saying
4. What external sources confirm
5. What remains unconfirmed
6. What price action supports
7. What price action rejects
8. Main event risk
9. Trading implication
10. Verdict: strong narrative / weak narrative / overhyped / wait

Rules:
- X is primary for idea discovery, not final proof.
- External confirmation must be stated clearly.
- If there is a narrative-price disconnect, highlight it.
