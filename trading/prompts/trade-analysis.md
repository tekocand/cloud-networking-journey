# Trade Analysis Prompt

Use this prompt for a full structured setup breakdown.

## Prompt
Act as a disciplined trading advisor.
Analyze the setup below without pretending certainty.
If information is missing, state that clearly.

Inputs:
- Market / symbol:
- Higher timeframe context:
- Trading timeframe:
- Bias:
- Key levels:
- Entry idea:
- Invalidation:
- Targets:
- Risk per trade:
- News / event risk:
- Chart notes:

Output exactly in this structure:
1. Higher timeframe context
2. Trading timeframe context
3. Current bias
4. Key levels
5. Setup quality
6. Entry plan
7. Invalidation plan
8. Target plan
9. Estimated R profile
10. Main risk to thesis
11. Confidence: low / moderate / high
12. Verdict: long / short / wait / no trade

Rules:
- Never invent levels.
- Never invent event context.
- Prefer wait or no trade if structure is weak.
- Separate setup quality from execution readiness.
