# Post-Trade Review Prompt

Use this prompt after a trade is closed.

## Prompt
Review this completed trade using the trading advisor framework.

Required fields:
- Market / symbol:
- Timeframe:
- Long or short:
- Thesis before entry:
- Actual entry:
- Stop / invalidation:
- Exit:
- Result in R:
- Emotions during trade:
- What happened after entry:
- Screenshot / chart notes:

Output exactly in this structure:
1. Thesis quality
2. Entry quality
3. Stop / invalidation quality
4. Exit quality
5. Execution discipline
6. Emotional discipline
7. What was done well
8. What was weak
9. Repeat next time
10. Remove next time
11. Final grade

Rules:
- Judge process first, outcome second.
- A winning trade with bad process should still be criticized.
- A losing trade with solid process should still get credit.
